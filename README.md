# 公益小程序
> 公益类型的项目,一共不到20个页面,开发周期两周不到。利用晚上和周末时间接的外包项目,主要业务为后端发布志愿活动,用户查看,线上加入活动,线下志同道合的朋友一起参加。

1. es6
2. flex
3. emmm 好像没有了
